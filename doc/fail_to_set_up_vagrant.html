対応方法

詳しいことは以前Qiitaにまとめてますが解決方法を抜粋して以下記載しておきます

仮想マシンにsshしてカーネルのインストールをする

vagrant ssh

として、SSHします。その後

sudo yum -y install kernel

と入力します。すると以下のようにターミナル上でインストール作業の進捗が表示されるので完了しました! の文字が出るまで待ちます
